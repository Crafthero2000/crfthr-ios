# AGENTS

- Всегда отвечай на русском.
- Коммит считается завершённым только после успешного git push.
- Перед предложением запускать GitHub Actions всегда выполняй:
  - git status --porcelain
  - git branch -vv
  - git log -1
  И только если всё чисто + последний коммит запушен — говори "CI можно запускать".
- Если push невозможен — объясни причину и запрети запуск CI.
